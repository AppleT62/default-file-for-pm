# 사전등록 — 실험 B: 온도 축(발견 S)의 조작 가능성

**등록 상태**: ☐ 미등록(제출 대기) — 등록 시 URL·일시 기입: __________
**시험 대상 (회차 72 재지정)**: "개인 규모에서, 산출 분포의 의미적 다양성은 온도 조작(판단 유예 + 무관 자극 노출 + 시간 압박 제거의 복합 조작)에 예측 방향(고온 → 다양성 증가)으로 반응한다."
**반증 카드**: 카드 ②′(EXPERIMENTS.md, 회차 72 재작성)

---

## 1. 연구 정보 · 가설

**핵심 가설 (H1, 방향)**: 동일 인물이 **고온 조건**에서 산출한 아이디어는 **저온 조건**보다 의미적 다양성이 높다.

**정직 고지 (사전 명기 — 절대 약화 금지)**: 이 조작(판단 유예 vs 평가 지시)은 브레인스토밍 문헌(오스본류)의 **표준 조작**이다. **이 실험은 카드를 켤 수만 있고 이론을 지지할 수는 없다** — 통과해도 확인되는 것은 차용 부품이지 "단일 온도 축이 네 규모에서 반복된다"는 고유 주장이 아니다. 고유 주장의 양의 검증에는 ① 3요소 조작의 요소 분해, ② 규모 간 이전 검증이 필요하며 이는 실험 D(미설계)의 몫이다. 규모 간 이전은 본 실험의 시험 대상이 **아니다.**

## 2. 설계

- **유형**: 사전등록, **피험자 내 교차**(within-subject, 순서 상쇄).
- **조작**: 동일 인물이 두 조건에서 아이디어 과제 수행 —
  - **고온 조건**: 판단 유예 지시 + 무관 자극 노출 + 시간 압박 제거.
  - **저온 조건**: 명시적 평가 기준 + 선별 지시.
- **순서 상쇄**: 조건 순서 무작위 배정(AB/BA 균형).

## 3. 표본 · 정지 규칙

- **표본**: n ≈ 66 (피험자 내 dz = 0.35, 검정력 80%, t-보정 포함 66–67명).
- **정지 규칙**: 사전 등록 표본 완성 시 종료.
- 등가성(무효과) 판정 필요 표본·등가 한계(TOST)는 등록 시 확정. 기본 표본으로 등가 판정 불가 시 그 사실을 실험 A와 동형으로 명시(비유의 = 판단 유보).

## 4. 변수

- **1차 결과변수 (판정 하중)**: 산출 아이디어의 **의미적 다양성** — 임베딩 분산 등 **사전 명시된 계산 절차**(모델·전처리·거리 지표를 등록 시 고정).
- **2차 결과변수 (α 보정·판정 하중 없음)**: 독창성/실행가능성 평정(맹검 평정자 2인, 신뢰도 보고).
- **예측 (사전 명시)**: 고온 → 다양성·독창성 ↑, 실행가능성 ↓. 저온 → 역방향.

## 5. 분석 계획

- 조건 간 의미적 다양성 차이 — 피험자 내 대응 비교(paired), dz 보고.
- 평정 지표는 2차(다중비교 보정), 평정자 간 신뢰도(ICC) 사전 기준.

## 6. 판정·반증 규칙 (사전 확정 — 회차 72 개정)

| 분기 | 조건 | 귀결 |
|---|---|---|
| ㉠ 1차 변수 **반대 방향** 유의 | 고온이 다양성을 유의하게 **감소** | **카드 ②′ 촉발** — 표적 하위 명제 폐기·발견 S 온도 성분 폐기·온도 축 의존 확장 항목 4건 재회부 |
| ㉡ 방향 일치 유의 | | 현행 값·라벨 유지(**상향 불가**) — 정직 고지 적용: 통과는 차용 부품 확인이지 이론 지지 아님 |
| ㉢ TOST 등가 입증 시에만 '무효과' | 대표본 요건 | '조작 가능한 축(개인 규모)' 주장 보류 강등 |
| ㉣ 기본 표본 비유의 | | 판단 유보 |

**촉발 확인**: 결과 공표 시 독립 판정 세션 회부 — 기계적 확인(사전등록 정합·유의성·방향)뿐, 회피 재량 없음. 조작 점검 실패 항변으로 촉발 무효화 불가(면역 장치 봉쇄).

## 7. 윤리·안전

- IRB/기관 승인. 사전등록 없이 수행된 결과는 카드 촉발/해제 근거로 불인정.
- 저위험 인지 과제 — 표준 브레인스토밍 절차 수준.
