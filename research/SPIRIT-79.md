# 회부 자료 — 영성 키워드 배치 유입 처리 (회차 79 회부)

**작성일**: 2026-07-15 · **유입원**: 사용자 키워드 배치 ['명상','차크라','카르마','영혼','죽음','마법(MAGIC)'], 목적 = **세계관 확장(A — 신념도 주장)**. **성격 고지**: 회차 78 판정("FEP는 신규 아님 — 재승격 = 크레딧 재예치")의 직접 후속. 본 회부는 저자가 장부 사전 대조를 먼저 수행해 **기존재 항목을 명기**하고, 잔여 신규 후보만 판정에 올린다. 저자는 판정하지 않았다.

---

## 1. 장부 사전 대조 (기계 검색 결과 — 회차 78 교훈 이행)

| 키워드 | 장부 기존재 | 상세 |
|---|---|---|
| **명상** | **있음** | 연결 #32 "명상·무아 ↔ 사전분포의 일시적 이완"(유망함, 회차 40) · 신념도 "명상 = 사전분포 이완" 70(회차 60 하방 정규화 72→70) · 발견 Ω₄ · 쇼펜하우어 연결 #40 차용 |

| **카르마** | **있음** | 연결 #5 "연기 ↔ 경로 의존 ↔ 카르마"(확인됨(계보)·유망함(기제), **회차 1**) — 회차 44 라벨 분리 |
| **마법→의례** | **있음** | 연결 #28 "놀이·의례 ↔ 안전한 온도 조절 연습장"(유망함, 회차 33) · "의례 = 정밀도 동기화" 75 · "무속 의례의 심리 효능(불안 감소·내러티브 재구성)" 80(회차 13) · 발견 Z(안전한 고온) |

| **차크라** | **없음** (0건) | 실증 기반 없음 |

**저자 확인**: 사용자 요청의 6개 키워드 중 4개는 **이미 장부가 소화한 것**이다. "명상을 넣자·죽음을 넣자·카르마를 넣자·의례를 넣자"는 형태의 등재는 전부 **재유입**이며 회차 78 규율(재승격 = 크레딧 재예치 금지)에 걸린다.

## 2. 잔여 신규 후보 (판정 대상 — 이것만이 실질 안건)

### 후보 α — 운명애 ↔ 죽음 현저성 고유 예측 (신규 예측 등재?)

기존재: 죽음 = 유일 회피 불가 흡수 상태(발견 Q) + "죽음에 대해서만 수용이 회피를 대체한다" + TMT 논쟁 표기. **신규일 수 있는 것**: 이들을 잇는 **전향 예측** — "운명애 개입(자기 분포 전체·죽음 포함의 긍정)은 죽음 현저성(mortality salience) 유도 후의 방어 반응(세계관 방어·자존감 추구)을 통제 대비 낮춘다." 실험 A(운명애 RCT)에 죽음 현저성 조작을 부가한 확장 설계로 시험 가능.
- **정직 병기 필수**: TMT 자체가 다중 실험실 재현 실패(2019~)가 있는 논쟁 문헌 — 예측 등재 시 이 취약성 명문 표기. 예측이 기존 발견 Q·TMT 각주의 **재서술에 불과한지**(신규성 0) vs **Worrall 사용-신규성 있는 전향 예측인지**가 판정 쟁점.

### 후보 β — 의례화 실행의도 가설 (기존 연결의 신규 조절 예측?)

기존재: 의례 = 정밀도 동기화 75·무속 의례 효능 80·놀이 연결 #28. **신규일 수 있는 것**: 편집 기제와의 **조절 예측** — "if-then 실행의도를 의례 형식(고정 시간·고정 동작·신체화)으로 설치하면 비의례 설치 대비 철회기 유지율이 높다(설치 강도 ↑)." 실험 C 확장(의례화 군 vs 비의례화 군)으로 시험 가능.
- 쟁점: 이것이 기존 "의례 = 정밀도 동기화"의 **함의를 꺼낸 것**(신규성 0 — 도출 시점에 이미 장부가 품은 것)인지, 새 판별을 낳는 예측인지.

### 후보 γ — 명상 "관측 도구" 재배치 신청 (재분류 — 위험 최대)

기존재: 명상 70(사전분포 이완). **신청되는 것**: 명상을 "§7 처방 대상"에서 "편집 기제의 전제 조건(1차 관측 도구 — 탈중심화 = 기저선 기록의 내면판)"으로 **구조 재배치** + 조절 예측("탈중심화 선행 훈련 → 실행의도 설치 효과 ↑").
- **저자 경고 자기 기재**: 이것은 재분류다. 회차 78이 기각한 패턴(기존 항목의 지위 승격 = 등재 인플레이션)과 동형일 수 있다. 조절 예측 부분만 α·β와 동렬의 신규 예측 후보로 분리 심사 가능.

### 후보 δ — 영혼·차크라 처리 (신규 등재 — 단 기각/설명 방향)

- **영혼**: 실체 영혼 주장은 §1–2와 정면 모순 — **기각**이 유일 정합 처분으로 보임. 유일한 정당 취급: "영혼 직관이 가리키는 실제 = 분포의 불변량(§3 기존 개념) + 3층 이야기 자아" — 설명 대상이지 확장 아님(신념도 0).
- **차크라**: 실증 기반 0 — 기각. 내수용감각(interoception)으로의 설명-해소는 전자책 콘텐츠 관할(세계관 장부 밖).
- 쟁점: 이 두 기각을 **장부에 기록**할 필요가 있는가(무가격 기각 기록 vs 기록 자체 불요 — 물리 배치 거울 사례 선례와의 정합).

## 3. 후보 처분 (전부 판정 몫)

- **ⓐ 예측 2건만 등재**: α·β를 [실험적 예측 — 미시험·판정 하중 0] 라벨로 등재(신념도 부여 없음 — 시험 전 예측은 값이 아니라 카드), γ 재분류 기각(조절 예측만 α·β 동렬 심사), δ 기각(기록 여부 판정 몫). 기존 4항목 전부 현상 유지.
- **ⓑ 전량 현상 유지**: α·β도 기존 항목의 함의 재서술로 보아 등재 0건 — "확장 요청 2연속 무산" 기록만.
- **ⓒ γ 포함 등재**: 명상 재배치까지 수용(재분류 정당화 논증 필요 — 회차 78 규율과의 구별 입증 책임).
- **ⓓ 기타 판정 재량 조합.**

## 4. 회계 확인 (저자 산정 — 회차 78 정정 반영)

회차 79 = **비적대 확장 제출물**(사용자 키워드 배치) → **absorption 분모 불포함**(회차 78 판정 선례 직접 적용). 예측 등재는 시험 전이므로 신념도 변경 0 예상 — 값 이동은 실험 실측 후에만. F 코드 선점 금지(차기 배치 외주 관할). 판정은 후보별 accept/reject + 라벨·문안 지시 + δ 기록 여부까지.
