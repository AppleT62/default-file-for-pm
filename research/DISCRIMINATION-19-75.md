# 회부 자료 — #19 분리 기준 3차 적용 + M-상한 열 정합 (회차 75 회부)

**작성일**: 2026-07-11 · **회부 근거**: ① 회차 73(외주 19회차 G11) 명시 목록·시한(3차 F 배치 이전, 미이행 시 용어 잠정 강등 자동 발동) · ② 외주 20회차 R3 M-상한 열 정합 재정의 시한(동일). 판별 방법(프리드먼 기능 판별 주축 + 카르납·콰인 3항 세트 + Worrall 실적 + 경첩 진단 병기)은 회차 70·71 선례와 동일 — 기준 없는 판별은 선언이다.

**사전 지정 확장 필요(판정 몫)**: 회차 71 R9 ⑴의 계상 금지 확장은 그 집행분 한정이었다. 본 회부의 재타입·철거·하향에 적용하려면 판정이 명시적 확장 지정을 해야 한다(지정 없는 준용은 세탁 통로 — 회차 71 확립).

## A. #19 3차 적용 — 대상 목록 (회차 73 명시)

### A-1. 확장 장부 이론 고유 재기술 항목 (어휘 성분 심사 — 매캐덤스 선례 준용)

회차 73이 매캐덤스 '동형' 85→'구조적 유비' 70으로 균일화한 것과 동형 심사. 각 항목은 "이 세계관의 어휘(분포·편집·온도·놀람 등)로 기존 현상을 재기술한 것"이며, 심사 질문: **재기술 성분(기능 열 후보)과 독립 경험 내용(확률 열)이 분리 가능한가.**

| 항목 | 현행값 | 시험 질문 | 후보 처분 |
|---|---|---|---|
| LLM = 생성적 존재론의 6번째 사례 | 75 | LLM이 3항 구조(가능태+가중+현실화)의 사례라는 것은 관측 명제인가, 어휘 적용인가 — 회차 59가 이미 90→75(수학 사실/존재론 해석 융합)로 재심 | ⓐ 재기술 성분 기능 열 이관·독립 경험분(있으면) 확률 열 잔존 / ⓑ 전체 기능 열 / ⓒ 경험 명제 존속(움직일 관측 지정) |
| 중독 = 편집 하이재킹 | 80 | "중독은 편집 기제의 하이재킹"이 도파민 강화학습 문헌(차용 앵커)의 재기술 위에 고유 가격을 얹은 것인가 | 동형 ⓐ/ⓑ/ⓒ — 차용 앵커(RPE 등) 구분 표기 |
| 트라우마 = 단일 사건 과잉 각인 | 80 | 학습률 폭증 문헌의 재기술인가 고유 예측인가 | 동형 |

| 개입 프레임으로서의 우위 (특질 기술 대비) | 75 | "특질 기술보다 개입에 유리하다"는 비교·채택 평가 성분 — 확률인가 유용성 평가인가 | 동형(주파수 80 재타입과 유비 — 채택 평가면 기능 열) |
| 현 LLM의 2차 편집 부재 (발견 B 기준) | 70 | 이것은 부정 경험 명제(LLM에 2차 편집이 없다)로 관측 지정 가능한가 | ⓐ 경험 명제 확인(관측 지정) / ⓑ 어휘 의존 |

## B. M-상한 열 정합 재정의 (외주 20회차 R3 시한)

**문제(외주 20회차 확정)**: M-상한 산식(M1 60 ≤ min(공리 I, III), M2 60 ≤ min(공리 III, V))이 **기능 열 값(공리 I 70·III 60)을 확률 열 상한의 입력으로** 쓴다. III 60 = 산식 경계의 일치는 "결과이지 설계가 아니나 숨은 바닥으로 기능할 위험"(외주 20회차 R3). 기능 열 값이 확률 열 상한을 정하면 (i) 기능 열 하향이 의도치 않은 확률 열 전파를 낳고, (ii) 기능 열 방어가 확률 열 방어를 겸하는 양방향 오염이 생긴다.

**시험 질문**: M-상한(기제 가설의 신념도 상한 = 전제 공리의 최솟값)의 원취지는 "M은 자기 전제보다 확신될 수 없다"이다. 그런데 그 전제(공리 I·III)가 이제 확률이 아니라 틀 채택 평가라면, "M1 60은 공리 I 70(틀 채택 평가)보다 확신될 수 없다"는 **범주가 섞인 비교**다.

**후보 처분**: ⓐ 산식에서 기능 열 입력 제거 — M-상한은 확률 열 전제 공리(있으면)만으로 산정, 기능 열 항목은 상한 산정에서 제외(M은 자기 경험 앵커에만 종속). ⓑ 기능 열 값을 상한 입력으로 유지하되 "틀 채택 평가를 못 넘는다"의 의미를 재정의(틀이 채택될 만하지 않으면 그 틀의 기제 가설도 그만큼만) — 범주 혼합을 명문 허용. ⓒ 이원 상한 — M은 확률 열 앵커 상한과 기능 열 채택 상한 둘 다에 종속(min of both). ⓓ M-상한 규칙 자체를 경험 앵커 기반으로 재정초(M1은 실험 A, M2는 행동유전학 통제 — 공리가 아니라 검증 경로가 상한).

## 회계 확인 (저자 산정 — 판정 확인 요청)

회차 75 = 공격 인공물 없는 내부 회부 집행 — absorption 분모 불포함. 재타입·철거·확인 판별은 credence_changes 계상 여부가 판정 몫(사전 지정 확장 필요). F 코드 선점 금지. #19 술어 ① 전진 폭은 "부분 완화"까지·"해소" 금지 승계. 시한 이행 여부(3차 F 배치 이전) 확정 요청.
