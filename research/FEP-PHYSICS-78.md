# 회부 자료 — 물리 자료(TCRF) 유입 처리: FEP 기제 연결 + 물리 장식 기각 (회차 78 회부)

**작성일**: 2026-07-12 · **유입원**: 외부 DEEP RESEARCH 문서(제미나이, "얽힘과 소산의 동역학 — TCRF"). 물리 키워드 9종(초끈·원자/전자·양자역학·특수/일반상대성·중력·열역학·엔트로피)을 인지·실존에 매핑해 자체 프레임(열역학적 인지-상대성)과 공리 3·프로토콜 4를 제시. **성격 고지**: 사용자가 "세계관 확장"을 요청. 본 회부는 이 자료의 어느 부분이 신념도를 벌어들이고 어느 부분이 흡수 표면·범주 오류인지 판정에 회부한다. 저자는 판정하지 않았다.

## 배경 — 이 프로젝트의 기존 급소 (판정 시 유지)

- (c) 판정 3연속(제미나이·GPT-5·그록): "반증 불가능한 유사이론". **물리-드레이핑은 이 (c)의 원형.**
- 실험 A·B 정직 고지: 차용 부품이 통과해도 세계관 고유 문법의 확증 아님(**감사 가능성 ≠ 반증 가능성**).
- 흡수 표면(#22)·등재 인플레이션: 개념 추가는 신념도를 못 벌 수 있음(재분류는 증거가 아니다).
- 세계관 핵심: "운명 = 조건부 확률 분포(**성향·빈도**), 자유 = 그 분포의 유계적 편집". "확률"은 성향적이지 양자적·물리적이 아님.

## 삼분 (저자 예비 분류 — 판정이 뒤집을 수 있음)

### A. 기제 연결 후보 (신념도 획득 가능?)

**자유에너지 원리(FEP)/능동적 추론(Friston)**: 실재 신경과학 연구 프로그램. 매핑 주장:
- 조건부 분포 ↔ 생성 모델(prior/기대)
- 2차 편집·실행의도 ↔ 능동적 추론(환경을 예측에 맞추는 행동)·예측오차 최소화
- 운명애(전 분포 긍정) ↔ 놀라움(surprise) 상한 관리

**시험 질문**: 이 매핑이 (i) M1류 **기제 가설**로 등재될 만큼 예측·판별을 낳는가, 아니면 (ii) 또 하나의 차용 부품/유추에 불과한가? 주의: FEP 자체가 "반증 곤란" 비판을 받는 프레임(등재 시 이 취약성 정직 병기 필요 — 반증 곤란한 틀로 반증 곤란성을 보강하는 자기모순 경계).

### B. 한정 사용 후보

**엔트로피·정보이론(볼츠만-섀넌 등가)·negentropy**: §3(시간)·"질서 유지 동기"의 한정 철학 논의로 서술 심화만(신념도 이동 없음)? 아니면 이마저 흡수 표면?

### C. 기각 회부 (물리 장식·범주 오류)

- **양자 인지**: 수학 형식(양자확률로 결합오류·순서효과 모델링)은 실재하나, 문서는 "결단 = 물리적 파동함수 붕괴"로 미끄러짐(범주 오류). 뇌=양자 주장 기각?
- **초끈이론 → 자아=진동**: 미확증 사변, 벌어들인 연결 0.
- **원자/전자 확률구름 → 인지 잠재성**: "확률" 단어 등가(양자확률 ≠ 성향확률).
- **특수/일반 상대성 → 주관적 시간·트라우마 곡률**: 물리적 시간지연·중력 ≠ 심리. '다중 시계'·'끌개'는 실재 신경과학이나 상대성 프레이밍은 덧칠.

**공통 급소**: **"확률"의 3중 미끄러짐**(양자확률 / 성향확률 / 베이즈 불확실성)을 명문으로 차단해야 하는가?

## 후보 처분 (전부 판정 몫)

- **ⓐ FEP만 기제 가설 등재** — 차용 부품 정직 표기·FEP 반증곤란성 병기·예측으로만 신념도. B는 서술 심화(신념도 0). C 전부 기각 + "확률 미끄러짐" 금지 조항 신설.
- **ⓑ 전량 은유로만**(book/전자책 각색), 세계관 신념도 무변동.
- **ⓒ 전량 기각** — 또 다른 흡수 표면(#22)·(c) 심화. 유입 자체를 반증 흡수로 회계.
- **ⓓ FEP 등재 + C 기각 + TCRF 전체를 '경쟁 유사이론 사례'로 9장(자기 재판)에 편입**(반면교사).

## 회계 확인 (저자 산정)

회차 78 = 외부 자료 유입 처리 → **absorption 분모 포함**(외부 자료 유입). FEP 등재 시 신념도 방향(±)·크기는 판정 몫. 물리 장식 기각은 신념도 무변동 예상. F 코드 선점 금지. 판정은 accept/reject(후보별) + 신념도 크기 + 신설 조항 문안 지시까지.
