# 회부 자료 — F 지수 시범 채점 집계·문턱 강건성 (회차 68 회부)

## 1. 코딩 결과 (원문 요약 — 코드 무수정)

- **P = 4**(회차 48·51·53·63) / **A = 17** / **N = 2**(49·66, 분모 불포함) — 23개 회차 전건 코딩
- P 가중: 48 비닫힘 1.0 · 51 S형 0.5(대안 C형 0.25) · 53 S형 0.5 · 63 S형 0.5
- ⚠ 모호 10건(코딩 세션이 대안 코드 병기): 47(A→P)·49(N→A)·51(P→A)·52(A→P)·56(A→N)·60(A→N)·62(A→N)·63(P→A)·64(A→N)·66(N→A)

## 2. 지수 산출 (규약 ⑥ — 분수 정확 계산)

| 구성 | F = P/(P+A) | 등급 | F′ = Σw/(Σw+A) | 등급 | **판정값(불리한 등급)** |
|---|---|---|---|---|---|
| **운용 창(⑥ 문언: 직전 10개 P/A 회차 = 56–65)** | 1/10 = 0.100 | 정체 | 1/19 ≈ 0.0526 | **퇴행** | **퇴행** |
| 전 구간(44–66, 참고) | 4/21 ≈ 0.190 | 정체 | 5/39 ≈ 0.128 | 정체 | 정체 |

주: 규약 ⑥의 지수 정의는 "직전 10개 P/A 회차 이동 창"이므로 **운용 값은 창 행**이다. 전 구간 값은 시범 채점의 보조 관측값. F′ 구성은 규약 ⑥에 명시(Σw/(Σw+A)) — 코딩 세션이 "미지정"이라 부기한 것은 코딩 세션에 규약 ⑥ 전문이 아닌 발췌만 제공된 탓(격리의 부수 효과, 코드에는 영향 없음 — 가중은 유형 병기로 산출 가능).

## 3. 강건성 산술 (⚠ 대안 코드 시나리오 — 전부 분수 정확)

| 시나리오 | 창 F | 창 F′ | **창 판정값** | 전 구간 판정값 |
|---|---|---|---|---|
| 기본 코딩 | 1/10 [정체] | 1/19 [퇴행] | **퇴행** | 정체 |
| S1: 63→A (B2·Worrall 충돌 반대 해소) | 0/10 [퇴행] | 0 [퇴행] | **퇴행** | 정체 |
| S2: 56·60·62·64→N (구조 변경 문언 독해) | 1/5 [정체] | 1/9 [정체] | **정체** | 정체 |
| S3: 상한 구성(47·52→P·49→A·구조변경→N) | 3/10 [퇴행 부재] | 2/9 [정체] | **정체** | 퇴행 부재 |
| S4: 51 가중 C형 0.25 | 1/10 [정체] | 1/19 [퇴행] | **퇴행** | 정체 |

- **어떤 시나리오에서도 운용 창의 판정값은 '퇴행 부재'에 도달하지 않는다**(범위: 퇴행–정체). 판정값을 올리는 유일한 통로는 ⚠ 대안 중 구조 변경 회차의 A→N 이동(S2·S3)이다 — 코딩 소견 1이 지적한 대로, **A→N 재분류는 분모 축소로 지수를 올린다**(문언 충돌의 처분 방향이 지수를 직접 움직임).
- 정정 부기(산술만): 코딩표 집계란의 상한 구성 수치 "F = 7/20 ≈ 0.350"은 정확 재계산 시 **전 구간 1/3 ≈ 0.333**이다(코드 구성은 코딩표 문언 그대로 사용, 산술 결과만 상이). 원문은 무수정 보존, 정정은 본 문서에만 기재.

## 4. 문턱 강건성 (규약 ⑦ — 등록 문턱의 검사; 문턱 이동 권한은 외주 재회부 전용)

- **하한 0.10의 민감 구간**: 기본 코딩에서 창 F′ = 1/19 ≈ 0.0526. 하한을 1/19 이하로 내리면 판정값이 퇴행→정체로 반전 — 반전 구간은 하한 ∈ (1/19, 1/10]. 등록 하한 0.10은 이 구간의 상단.
- **경계 정확치**: 창 F = 1/10은 하한과 정확히 일치(규약 ⑦: 0.10은 정체 구간 포함, 반올림 없음) — 창 내 코드 1건 반전으로 등급이 움직이는 위치다(63→A면 0/10 퇴행).
- **상한 0.25**: 어떤 시나리오도 운용 창에서 상한을 다투지 않음(최고 3/10, 단 그 시나리오의 F′가 2/9로 끌어내림) — 상한은 본 표본에서 미시험.
- **정체 연속 2회 = 퇴행 전환 조항**: 본 시범 채점은 단일 시점 관측이므로 미발동 — 상시화 시 차기 창부터 카운트 개시 여부는 판정 사항.

## 5. 코딩 세션이 회부한 규약 문언 계쟁 2건 (처분은 판정 몫 — 규약 ⑪: 채점 데이터 열람 후 개정은 외주 재회부 전용 + 항구 기록. 본 회부가 그 재회부다)

1. **B3 vs ③A 충돌(공리 강등 회차 56·64)**: "구조 변경은 N"과 "라벨 강등·주장 축소는 A"가 정반대 코드를 지시. 처분 방향이 지수를 직접 움직임(§3) — A 유지가 저자에게 불리한 방향.
2. **B2 vs ②(iii) 우선순위(회차 63)**: 자기 구속 규칙의 최초 실채점 결과가 촉발 공격의 닫힘에 사용된 경우의 P/A. 코딩 세션은 B2 우선(P)을 택했고 대안 A를 병기.
