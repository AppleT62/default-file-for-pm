# 회부 자료 — 실험 B·C 요인 설계 재작성 + 카드 ②′ (회차 72 회부)

**작성일**: 2026-07-11 · **맥락 사실**: ① 실험 B의 현행 표적("공리 V·발견 S의 축 승격 폐기", "한 규모에서라도 반대 효과")은 3중으로 실효 — 공리 V 축약(회차 64)·카드 ② 효력 철회(회차 69: 조작 미정의+표적 상실)·스케일 불변성 60 철거(회차 70). ② 실험 B의 정직 고지(회차 50)가 이미 자백: "이 실험은 카드를 켤 수만 있고 이론을 지지할 수는 없다 — 고유 주장의 양의 검증에는 요소 분해 설계·규모 간 이전 검증이 필요하다 — **미설계 상태로 등재**." ③ 실험 C에는 중단·의뢰 기준이 없다(보고서 #5 G6 — §9 게이트는 회차 69에 인라인됐으나 실험 설계 자체의 중단 규칙은 부재). ④ #21의 닫힘 술어 전반부 = "kill-condition 카드를 현실적으로 관찰 가능한 촉발 조건으로 재작성 + 최소 한 장 실시험."

## Q1 — 실험 B 재설계 (표적 재지정 + 요소 분해)

**현행 유지 불가 사실**: 판정 규칙의 촉발 대상(공리 V 폐기·발견 S [은유] 강등)이 회차 64·70–71 이후 존재하지 않거나 이미 강등 상태다.

**선택지**:
- ⓐ **2×2×2 피험자 내 요인 설계로 전환** — 3성분(판단 유예 / 무관 자극 노출 / 시간 압박 제거)을 독립 조작, 표적을 규모 특정 하위 명제로 재지정: "개인 규모에서, 산출 분포의 다양성·독창성은 온도 성분 조작에 예측 방향으로 반응한다." 성분별 효과 분리(정직 고지의 요소 분해 요구 이행). 비용: 조건 8개 — 피험자 내 완전 교차는 회기 부담 큼(축소형: 주효과 중심의 부분 요인 배치 가능성 병기).
- ⓑ **현행 단일 요인 유지 + 표적만 재지정** — 설계 불변, 판정 규칙의 촉발 대상을 "발견 S의 '조작 가능한 축' 주장(개인 규모 한정)"으로 교체. 비용 최소·요소 분해 미이행(정직 고지의 자백 존속).
- ⓒ ⓐ의 완전판이 부담이면 **2단계**: 1단계 = ⓑ(표적 재지정 즉시), 2단계 = 요소 분해를 실험 D(연구자용)로 이관.

**카드 ②′ 재작성 문언 후보(판정 몫 — #21 술어 전반부의 이행 여부 판정 포함)**: "사전등록된 온도 조작 실험(개인 규모)에서 1차 변수(의미적 다양성)가 예측과 **반대 방향**의 유의 효과를 보이면 → 발견 S의 '조작 가능한 축' 주장 폐기 + 편집 축 어휘의 '온도' 성분을 [은유] 강등, 편집 기제 65의 온도 관련 서술 연쇄 갱신." — 촉발 조건이 현실적으로 관찰 가능(사전등록 실험 1회)·자동 발동. 저자 부기: 카드 ②′ 등재는 카드 ② 철회(반증 태세 악화)의 **부분 회복**이며 §10에 "재작성"으로 기록(회차 69의 "개선 서술 금지"는 철회 자체에 대한 것 — 재작성의 서술 방식은 판정 지정 필요).

## Q2 — 실험 C 재설계 (성분 관찰 + 안전 중단 규칙)

**선택지(성분)**:
- ⓐ **단계 도입 설계(A–B₁–B₂–A′)**: 기저선 2주(기록만) → B₁ 3주(+실행 의도) → B₂ 3주(+수면 게이트 적극 운용/중요도 낮추기) → 철회 2주. 성분 증분의 관찰(탐색 표기 — N=1이므로 판정 하중 없음, 판정은 여전히 전체 개입 vs 기저선·철회 복귀).
- ⓑ 현행 ABA 유지(성분 분해는 실험 D로 전량 이관).
- 공통 사실: N=1에서 성분의 확증적 분리는 불가능 — G2의 확증적 분리는 실험 D 관할임을 어느 선택지든 명기 필요.

**안전 중단 규칙(G6 — 선택지가 아니라 문언 확정 요청)**: 후보 문언 — "주간 체크에서 기분·수면·기능 중 2개 영역 이상의 악화가 2주 연속 관찰되면 **개입 중단 + PROTOCOL 임상 게이트의 전문 평가 절차로 이행**(자동 발동·본인 재량 불가). 기록 부담 문항(#24)이 2주 연속 상위 앵커면 기록 부하를 일 1건으로 감축(중단 아님 — 부담 완화). 중단 발생은 실패가 아니라 안전 게이트 작동으로 기록하며, 그 자체가 #24의 관측 자료다."

**CACE·완수 편향**: 회차 61 주석(ITT/ABA 판정·완수 조건부는 보고 전용) 존속 확인 + 연구자용 실험 D에 CACE/주 층화 명시 여부.

## Q3 — 실험 D 신설 여부 (연구자용 — 프로토콜 성분 분해 MOST + CACE)

후보 골격(등재만·미설계 명시): "66일 프로토콜 패키지의 성분(조건부 기록 / 실행 의도 / 0층 게이트 / 중요도 낮추기)을 MOST 준비 단계 프레임으로 요인 분해, 1차 변수 = 표적 행동 빈도(행동 지표), CACE·주 층화로 완수 편향 처리, 안전·위해 모니터링 사전 등재(중단 규칙·부작용 문항)." — G2의 "세 주장 분리" 중 "이론 고유 기제"의 확증 경로. 선택지: ⓐ 등재(미설계 골격 — 실험 A 표본 산정과 동급의 구체화는 차기), ⓑ 기각(실험 A–C 완료 전 등재는 목록 인플레), ⓒ 등재하되 #8(실험 미실행) W에 부속.

## Q4 — 문서·카드 회계

- §10 카드 문단: 카드 ②′ 등재 시 "2장 → 1장(② 철회) → 2장(②′ 재작성, 회차 72)"의 이력 명시 서술 — 정직 서술 문언 판정 필요(철회 기록 불소멸).
- #21 W/S: 재작성 1장 이행 기록(닫힘은 "최소 한 장 실시험"이 남음 — 전반부만).
- EXPERIMENTS.md 실행 상태 표·실험 B 표적 문언·실험 C 설계 절 전면 개정.
- 실험 B 판정 규칙의 "공리 V·발견 S" 참조 전건 청소.

## 회계 확인 (저자 산정 — 판정 확인 요청)

회차 72 = 공격 인공물 없는 내부 회부 집행 — absorption 분모 불포함. 설계·카드 등재는 증거가 아니다 — credence_changes 0 예상(카드 ②′ 등재·실험 D 등재 모두 신념도 무변동). F 코드 선점 금지 — 차기 배치 외주 관할(본 회부·판정문에 코드 유도 서술 금지).
