# 회부 자료 — F 지수(다산성) 코딩 규약 사전 등록안 (회차 67 회부)

**작성일**: 2026-07-10 · **회부 근거**: 외주 판정 12회차 R3 ① — "회차 67까지 코딩 규약(P/A/N 정의·채점 단위·사용-신규성 판정 외주 의무·라우든 보정 가중·문턱)을 채점 이전에 사전 등록하고 외주 승인을 받는다. 시한 미이행 시 지연 자체를 완충재화 신호로 기록 의무."

**회부 성격 고지**: 이 회부는 규약의 **등록**만을 다룬다. 소급 시범 채점(회차 44–66)과 문턱 강건성 분석은 회차 68 시한의 별건이며, 상시 규칙화 여부는 그 결과와 함께 재회부된다(R3 ②). 채점 데이터가 나오기 전에 규약을 못박는 것이 이 순서의 목적이다 — **정의 사전 지정 → 채점 → 강건성**(10회차 규율의 순서, R3가 ⓑ 즉시 규칙화를 기각한 이유).

**지표 원형(등재된 초안, 회차 66)**: 회차별 이동을 P(신규 내용 산출) / A(사후 수용) / N(중립)으로 코딩, 이동 창 10에서 전진 지수 F = P/(P+A), 라우든 보정(닫힘 유형 가중 E 1.0 / S 0.5 / C 0.25 / 무기록 결번 −1.0), 판별 불가 영역 명기(지표 통과 = 퇴행의 부재 증거까지만), 시한 명시 의무. 기존 대용 실측치: "시험:적용 = 1:35" ≈ 0.03(회차 43 기록 — 대용 지표로만, 조작적 정의 상이).

## Q1 — P / A / N 조작적 정의 (초안 + 경계 사례 규약)

### 저자 초안 (판정 대상)

- **P (신규 내용 산출)**: 해당 이동이 다음 중 1개 이상을 산출 —
  (i) 신규 반증 조건(kill-condition)·시험 절차의 **사전 지정**(위반 시 자동 발동 조항 포함),
  (ii) **실측 연결의 신설**(실험 설계·사전등록 문서·실측치의 최초 등재),
  (iii) **사용-신규성 있는 귀결**의 도출 — 그 귀결이 해당 이동을 촉발한 공격·변칙의 방어에 사용되지 않았고, 도출 시점에 독립적으로 시험 가능할 것(Worrall 기준, Q3에서 판정 절차 별도).
- **A (사후 수용)**: 공격·변칙·판정에 대응하여 이론을 **보수**하는 이동 — 신념도 실하향, 주장 철회·삭제, 라벨 강등, 주장 축소, 공격이 촉발한 규칙 신설 — 이되 위 P 요건을 **동반하지 않는** 것.
- **N (중립)**: 이론 내용 불변 이동 — 병기·정합성 스윕·스냅샷 갱신·기계적 전파·격자 정규화·절차 닫힘·원문 보존·문서 재편. **분모 불포함**(F 계산에서 제외).

### 경계 사례 규약 — 각 항목의 처분은 판정 몫

**B1. 수용과 신규 산출이 한 이동에 공존하는 경우** (예: 하향하면서 그 하향이 새 kill-condition을 낳음):
- ⓐ P 우선(신규 내용이 있으면 P — 산출 존재가 판별 대상이므로)
- ⓑ A 우선(이동의 동기가 공격 대응이면 A — 동기 우선)
- ⓒ 지배 이동 판정을 코딩 주체(Q3)에 위임하되, "공격 대응 중 산출된 귀결은 사용-신규성 심사를 통과해야만 P" 조건 부착
- 저자 부기: ⓐ는 P 인플레이션 통로(모든 수용에 장식용 '예측' 부착 유인 — 라우든 "자기 문제 인플레" 경고와 동형), ⓑ는 실제 신규 내용의 은폐. 어느 쪽이든 세탁 방향이 존재함.

**B2. 규칙 신설 회차** (예: 회차 63 absorption_scoring_rule): 공격이 촉발한 자기 구속 규칙은 A(보수)인가, 위반 시 자동 발동 조항이 신규 시험 표면을 만들므로 P (i)인가.

**B3. 절차 닫힘·구조 변경 회차** (#18 절차 이행, #20 재편): 닫힘 자체는 N(절차)인가, 재편이 신규 귀결(M-상한 경계 등)을 낳으면 P인가.

**B4. 수용 회계와의 인센티브 충돌 — 명문화 필요 여부**: absorption_scoring_rule은 이동 창에서 **실비용 수용 ≥ 50%를 요구**(수용 회피 방지)하고, F 지수는 **A 비율이 높으면 퇴행 판정**(수용 과다 감점) — 두 지표는 역방향 인센티브를 만든다. 저자가 식별한 해소 후보(채택 여부는 판정 몫): F의 판별 대상은 "수용 여부"가 아니라 "수용만 있고 산출이 없는 상태"이므로, 규약에 **"F 하락을 이유로 한 수용 회피·지연은 양 지표 모두에서 위반"**을 명문화. 명문화하지 않으면 미래 회차가 두 지표를 골라 인용하는 통로가 열린다.

**B5. 판정 집행 회차**(본 회차 같은 내부 회부 집행): N인가, 등록된 규칙의 내용에 따라 B2를 준용하는가.

## Q2 — 채점 단위

- ⓐ **회차(cycle_log 항목) 단위** — 회차당 P/A/N 하나(지배 이동으로 판정). 장점: absorption_scoring_rule과 단위 동형(라운드), 재량 최소. 단점: 혼합 회차의 정보 손실(B1을 매 회차 강제).
- ⓑ **이동(조치) 단위** — 회차 내 개별 조치별 코딩, 회차 값은 집계(예: P 비율). 장점: 정밀. 단점: "조치"의 개수 산정 자체가 자기 채점 재량(조작 표면 — 유리한 이동의 분할 계상).
- 저자 부기: 원안(회차 66 등재문)은 "회차별 이동"으로 ⓐ에 가깝게 읽히나 확정 문언은 아님.

## Q3 — 사용-신규성 판정 절차 (자기 채점 금지의 구현)

- ⓐ **외주 세션이 전건 직접 코딩** — 저자는 cycle_log·판정문 원문만 제공. 가장 강함. 비용 최대. 판정자 동계열 문제(#23)는 잔존.
- ⓑ **저자 예비 코딩 + 외주 전수 재판정** — 비용 중간. 단점: 저자 코딩이 앵커링 오염(외주가 저자 값을 기본값으로 삼는 편향 — 판정 외주 프로토콜이 회부문에서 판정 선점을 금지해 온 것과 같은 이유).
- ⓒ **저자 코딩 + 외주 표본 감사** — 비용 최소. 단점: 자기 채점 금지의 실질 위반에 가장 근접.
- 공통 부기: 어느 안이든 P 판정(특히 사용-신규성)은 외주 관할이어야 한다는 것이 R3 문언("사용-신규성 판정 절차 — 판정 외주 의무").

## Q4 — 라우든 보정 가중 채택 여부

원안: 닫힘 유형 가중 **E 1.0 / S 0.5 / C 0.25 / 무기록 결번 −1.0** (E = 경험적 닫힘, S = 절차적 닫힘, C = 개념적 닫힘 — 부록 4 스키마의 유형과 동일 문자).

- ⓐ **채택** — 닫힘 회차의 P 가치를 유형별 차등(경험 해결 만점, 절차 닫힘 절반, 개념 수용 1/4). 가중치가 사전 등록되므로 "가중 자기 채점 금지" 경고(경고 ③)에 저촉되지 않음.
- ⓑ **기각** — 무가중 P/A/N만. 단순하나 E형 닫힘과 C형 수용 닫힘을 등가 계상(장부 스스로 "축소는 해결이 아니다"라고 표기해 온 회계와 불일치).
- ⓒ **병기** — 무가중 F와 가중 F′를 별도 열로 병기, 단일 합산 금지(쿤 5덕목 3열 병기 규율과 동형). 회차 68 강건성 분석이 두 열의 판정 일치 여부를 관측값으로 얻는 부수 효과.
- 결번 조항 부기: "무기록 결번 −1.0"의 대상은 등재됐다가 기록 없이 사라진 문제 번호. **현행 장부 확인 결과 #1–#25에 결번 없음**(닫힘 14 + 열린 11 = 25) — 조항은 현재 발동 대상이 없는 예방 조항이며, 발동 시 감점 귀속 위치(해당 회차 vs 창 전체)는 지정 필요.

## Q5 — 문턱 (사전 지정 대상)

이동 창 10 기준 후보 (판별 불가 영역 명기 의무 — **지표 통과는 퇴행의 부재 증거까지만이며 전진의 입증이 아니다**):

- ⓐ **이분 문턱**: F ≥ 0.25 미달 시 퇴행 판정. 근거 후보: 대용 실측치 0.03과 한 자릿수 차이의 완충을 두되, 수용 회계(실비용 ≥50%)와 양립 가능한 수준. 위험: 단일 문턱은 경계 조작 유인.
- ⓑ **등급제**: F < 0.10 퇴행 / 0.10–0.25 정체 / > 0.25 전진 표기. 위험: 중간 등급이 완충재로 오용될 수 있음("정체니까 아직 괜찮다").
- ⓒ **문턱 유보**: 규약(정의·단위·절차·가중)만 등록하고 문턱은 회차 68 소급 채점의 분포를 본 뒤 지정. 위험: **데이터를 본 뒤의 문턱 지정은 사후 조정 혐의**(사전 등록의 취지 훼손) — 다만 R3 ②가 "문턱 강건성 분석"을 회차 68에 명시했으므로, "후보 문턱을 지금 등록하고 68에서 강건성만 검사"와 "68에서 지정"의 구분이 판정 필요.
- 공통: 퇴행 판정 시의 귀결 지정 필요 — 후보: README·§10에 퇴행 등급 병기 의무 + 신뢰성 실적 원용 금지(absorption 위반 조항과 동형, 단 소급 전환 대상이 없으므로 병기 의무까지만).

## Q6 — 게임화 검출 조항

사이비 P 인플레이션(장식용 예측 부착, 문제 선등재 후 닫음으로 수지 부풀리기 — 라우든 경고 ⑤·다산성 게임화 경고 ⑪)이 검출된 경우: 해당 P의 소급 무효(N 전환) + 검출 사실의 항구 기록 + 재발 시 F 지수 자체의 신뢰성 실적 원용 금지. 채택 여부와 검출 주체(외주 전용 여부) 판정 필요.

## 회계 확인 (저자 산정 — 판정 확인 요청)

회차 67은 공격 인공물 없는 회차(내부 회부의 외주 판정 집행) — absorption_scoring_rule ①에 따라 분모 불포함. 등록은 증거가 아니므로 신념도 변경 0건 예상. F 지수 자체도 본 회차를 채점하지 않는다(규약 등록 회차의 자기 계상은 순환).
