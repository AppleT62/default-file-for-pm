# 회부 자료 — axiom_linkage_rule 열 정합 재정의 (회차 77 회부)

**작성일**: 2026-07-11 · **회부 근거**: 외주 21회차 R5 목록 #1("axiom_linkage_rule의 기능 열 공리 적용 잔여 — M-상한과 동형의 열 혼합, M-상한은 재정초했으나 이 잔여는 미처리")·#2("연동·전파 규칙 일반의 열 구분 정합"). **성격 고지**: 본 회부는 **정합성 수리**이지 재분류가 아니다 — 어떤 항목도 기능 열로 새로 옮기지 않으며(기능 열 15항목 불변), 이미 기능 열인 공리에 확률 열 규칙이 잘못 적용되던 것을 끊을 뿐이다. 따라서 **#26(값 보호 아키텍처 혐의) 위험 없음**(팽창 0). 신념도 변경 0건 예상.

---

## 문제 (외주 20·21회차가 M-계열에서 이미 확정한 것의 공리판 잔여)

`axiom_linkage_rule`의 **회차 51 도입 조항**: "공리의 신념도 상한은 그 공리를 지지하는 연결들의 판정 등급이 결정한다 — 확인됨(기제) 포함 시 85, 확인됨(계보/현상)+유망함(기제) 시 75, 유망함뿐이면 70. **확률은 아래(연결)에서 위(공리)로만 흐른다.**"

이 규칙은 확률 열 논리다 — "관측·연결 증거가 공리의 참 확률 상한을 정한다." 그런데 공리 I·II·III·V·VIII는 회차 70·71·75에 **기능 열(틀 채택 평가)로 판별 확정**됐다. 기능 열 값은 관측이 아니라 **손익 회계(F 배치·개입 판독)와 외주 재회부**만 움직인다(ledger_identity). 따라서 "연결 판정 등급이 공리의 상한을 정한다"는 조항을 기능 열 공리에 적용하는 것은:

- **M-상한 재정초(회차 75)가 끊은 것과 정확히 동형의 범주 혼합** — 거기서는 "경험 명제 M이 틀 채택 평가(공리)보다 확신될 수 없다"를 끊었고, 여기서는 "틀 채택 평가(기능 열 공리)가 관측 연결 등급에 종속된다"를 끊어야 한다.
- **양방향 오염 통로**: (i) 확률 열 연결의 등급 변동이 기능 열 공리 값에 전파되는 통로, (ii) 기능 열 공리 값이 여전히 확률 열 규칙의 관할인 것처럼 보이는 회계 분열.

확률 열에 남은 공리는 **IV 하나**뿐이므로, 회차 51 조항의 정당한 관할은 이제 공리 IV로 축소된다.

## 시험 질문

회차 51 조항의 원취지("공리는 자신을 지지하는 증거보다 확신될 수 없다")가 공리가 기능 열이 된 지금 어떻게 보존되는가 — 기능 열 공리의 "확신"은 참 확률이 아니라 틀 채택 평가이고, 그 상한은 증거 연결이 아니라 손익 회계가 정한다(M-상한 재정초와 동일 논리).

## 후보 처분 (전부 판정 몫)

- **ⓐ 기능 열 공리 적용 제외 (M-상한 재정초 동형)**: 회차 51 조항은 확률 열 항목 전용 — 확률 열에 남은 공리 IV에만 적용. 기능 열 공리(I·II·III·V·VIII)의 값은 손익 회계·외주 재회부만 정한다(연결 등급 상한 비적용·전파 차단). 재도입은 외주 재회부 전용.
- **ⓑ 재해석 (범주 혼합 명문 허용)**: 연결 판정 등급을 기능 열 공리의 "채택 근거 강도"로 재해석해 조항을 유지. (M-상한에서 ⓑ가 기각된 이유와 동형 — 서열 비교 잔존.)
- **ⓒ 유지 (문제 아님)**: 회차 51 조항을 기능 열 공리에도 그대로 적용.
- **ⓓ 이원 조항**: 확률 열 공리는 연결 등급 상한, 기능 열 공리는 손익 회계 상한 — 별도 명문.

## 전파 규칙 정합 (R5 #2 — 부속)

회차 61 "III←편집 기제 전파" 유형(한 항목 하향이 연동으로 다른 항목에 전파)이 기능 열 항목에 걸리는 경우의 처리. 후보: 기능 열 항목 간 전파는 손익 회계 관할로 통일(확률 열 연동 전파와 분리), 또는 현행 유지. 판정 몫.

## 회계 확인 (저자 산정)

회차 77 = 공격 인공물 없는 내부 회부 집행 — absorption 분모 불포함. **신념도 변경 0건 예상**(규칙 정합 수리 — 어떤 값도 이동하지 않음, 기능 열 팽창 0). F 코드 선점 금지. 이것은 이원 회계 분리(회차 71)의 마지막 구조적 잔여 정리이며, 이후 저장소의 규칙 층에는 열 혼합 잔여가 없어야 한다(판정 확인 요청).
