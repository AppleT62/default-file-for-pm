# 시간·차원·운명 검증 루프

미래·자유·운명에 관한 세계관을 42회차·12기에 걸쳐 검증한 기록과 그 실천 도구.

> **한 문장:** 운명은 나의 조건부 확률 분포, 자유는 그 분포의 유계적 편집,
> 의미는 그 편집에 방향을 부여하는 최상위 지향 — 셋은 한 구조의 세 높이다.

## 파일 안내

| 파일 | 내용 |
|---|---|
| [`index.html`](index.html) | **대시보드** — 31개 부의 인터랙티브 보고서(JS 애니메이션·워크시트·네트워크 지도). 브라우저로 열어서 봄 |
| [`SYNTHESIS.md`](SYNTHESIS.md) | **종합 논고** — 회차별 기록이 아니라 도달한 결론을 10장의 완결된 글로 재구성 |
| [`PROTOCOL.md`](PROTOCOL.md) | **66일 실전 워크시트** — 이론을 자기 삶에서 검증하는 기록 템플릿 |
| [`data/worldview.json`](data/worldview.json) | **데이터 원본** — 신념도·연결 33개·발견 32개(A–Ω₅)·공리 8개·회차 로그 42행 전체 이력 |

## 최종 상태

- 12개 기, 42회차, 연결 33개, 발견 32개(A–Ω₅), 공리 8개
- 핵심 신념도는 1기 수렴(회차 7) 이후 42회차 내내 불변 — 견고성의 증거
- 반증 대기 상태로 봉인. 급소(반증 조건)는 CAPS·실행의도 다리 (대시보드 22부)
