# 4부 · 개념 네트워크 확장 아이디어

> 대상: `index.html` 4부(개념 네트워크) 섹션.
> 작성 시점: 회차 79 기준. 성격: **설계 제안서** — 검증 루프의 판정 대상이 아니며 신념도 무부착.

---

## 0. 진단 — 네트워크는 회차 3에 얼어붙어 있다

| 항목 | 4부 네트워크 (현재) | 장부 (`data/worldview.json`, 회차 79) |
|---|---|---|
| 연결 | 약 10개 (SVG 하드코딩) | **43개** |
| 발견 라벨 | A, B, C, E | **34개 (A–Ω₇)** |
| 절단 표시 | 범심론→벨 정리 1건 | 불성립 2건 + 강등·철거 다수 |
| 층위 | 단일 허브(주파수) 방사형 | 0층(육체)~3층(실천) + 이원 회계(확률 열/기능 열) |

네트워크에 담긴 것은 전체의 약 1/4이다. 13기 독일 철학 3인 적수전(연결 #34–41),
14기 도덕운·아비투스(#42–43), 4~12기의 삶의 현상 확장(#12–33)이 전부 빠져 있다.
또한 SVG가 손으로 그린 그림이어서 회차가 진행될 때마다 네트워크가 낡아간다 —
열린 문제 **#27(처방 표면 전파 술어 부재)** 이 지적하는 `index.html` 미전파의 실사례.

## 1. 데이터 기반 렌더링 — 가장 구조적인 확장 【추천 1순위】

`worldview.json`의 `connections` 배열에는 이미 `id` · `name` · `verdict` · `cycle_added`가 있다.

- **JSON에서 직접 그리기**: JS로 43개 연결을 자동 배치·렌더. 이후 회차에서 연결이
  추가되면 네트워크가 저절로 따라온다. #27의 근본 해결이기도 하다 (표면 5곳 기계
  대조에서 index.html이 장부와 같은 원천을 공유하게 됨).
- **회차 슬라이더**: `cycle_added` 필드로 "회차 1 → 79까지 네트워크가 자라나는"
  타임라인 재생. 절단(범심론, 힘에의 의지)과 강등(죽음-지평, 신점)의 순간도 함께
  보이면 '검증 루프'라는 문서의 정체성이 시각적으로 완성된다.
- **노드 클릭 → 팝업**: 판정문·신념도(또는 기능 열 값)·등재 회차·관련 발견 라벨.
- **판정 등급 필터**: 확인됨만 / 유망함 포함 / 절단·철거 포함.

## 2. 허브-스포크 → 층위 밴드 재설계

연결 43개를 단일 허브 방사형에 다 넣으면 스파게티가 된다. 배치 원리는 문서 자신이 이미 제공한다.

- **발견 I ("모든 발견은 층간 다리였다")**: 0층(육체) – 1층(물리) – 2층(기술) –
  3층(실천)의 가로 밴드로 노드를 배치하고, 엣지가 층을 가로지르는 모습 자체를
  보여준다. 발견 T(0층)는 현재 네트워크에 노드조차 없다.
- **발견 U (스케일 불변성)**: 개인·관계·조직·문명 동심원 링으로 사회 스케일 노드
  (아비투스 #43, 제도 #23, 세인 #12, 양육 #24, 도덕운 #42)의 자리를 만든다.
- **급소 강조 (발견 H)**: CAPS–실행의도 다리는 이 세계관의 유일한 경험적 반증
  관절인데 현재 아무 표시가 없다. "여기가 끊어지면 전체가 무너지는 지점"으로
  시각적 강조(맥박 표시 + 반증 카드 링크)를 부착한다.

## 3. 시각 문법을 장부의 판정 체계에 맞게 세분화

현재 범례 4종(확인/유망/동형/절단)으로는 장부의 판정을 다 표현하지 못한다.

- **계보/기제 분리 판정**: "확인됨(계보)·불성립(기제)" 같은 복합 판정(#5, #34,
  #43)이 다수 — 이중선 또는 반색(半色) 엣지로 구분.
- **차용(지반 단절) 상태**: 죽음-지평(#14), 미적 관조(#40)처럼 "빌려왔지만 지반이
  다름을 명시"한 연결의 전용 선 스타일.
- **이원 회계 반영(회차 71~)**: 확률 열 노드와 기능 열 노드의 시각 구분. 공리
  5개(I·II·III·V·VIII)가 기능 열로 재타입된 지금, 모든 노드를 같은 문법으로
  그리면 열린 문제 **#19가 기소한 범주 착오를 시각적으로 반복**하는 셈이다.
  정직성 차원에서 사실상 의무적인 확장.
- **절단 묘지 확대(이력 보존형)**: 범심론 외에 힘에의 의지(#41), GFP 폐기(#7),
  철거 항목(형이상학적 실재·성장 블록·시뮬레이션 묶음)까지 — "무엇이 죽었는가"의
  기록 보존.

## 4. 총괄 지도 + 클러스터 서브맵 분할

한 장에 다 넣는 대신 총괄 지도 1장 + 상세 지도 3~4장으로 나눈다.

| 서브맵 | 포함 연결 | 배치/서사 축 |
|---|---|---|
| 독일 철학 3인 (13기) | #34–41 | '적수 등판형 시험' 서사 — 생존/절단/차용의 대비 |
| 삶의 현상 (4~12기) | 사랑 #18 · 창의성 #21 · 중독 #27 · 놀이 #28 · 트라우마 #29 · 꿈 #30 · 명상 #32 · 예술 #33 등 | 발견 Ω₂(온도×입력 2×2)를 좌표축으로 — 배치 자체가 이론 설명 |
| 사회 스케일 (14기 포함) | 세인 #12 · 제도 #23 · 양육 #24 · 도덕운 #42 · 아비투스 #43 | 발견 U 동심원 |
| 형식과학 | LLM #17 · 에르고딕성 #19 · 플리슨 #13 · 콕스 정리(발견 G) · 알로스타시스 #22 | 층간 다리(발견 I) |

## 5. 새 개념 노드 후보 (등재 프로토콜 준수 전제)

네트워크에 바로 그리는 대상이 아니라 **검증 루프에 후보로 선등재**하는 대상.
이미 계획된 세계 편 확장 6종(`prompts/EXPAND-01~06`: 협력·경제·진화·기술-주의·권력·방법)과
겹치지 않는 후보만 나열한다.

1. **강화학습의 탐색-활용 트레이드오프 ↔ 온도 편집(발견 S)** — LLM 온도 연결(#17)은
   있는데 RL 다리가 없다. 가장 자연스러운 공백.
2. **카너먼 이중 프로세스 ↔ 1차/2차 편집** — 계보 후보.
3. **프래그머티즘(제임스·듀이의 습관론) ↔ if-then 분포** — 퍼스 준칙이 이미 #19
   도구로 수입돼 있으므로 계보 연결 승격 검토 가치.
4. **스피노자 코나투스·정서론 ↔ 분포의 동력 항** — 힘에의 의지 절단(#41) 자리에서
   '동력 항' 문제를 재시험할 적수. 13기 적수 등판 형식 재활용.
5. **메를로퐁티 몸틀 ↔ 0층(발견 T)** — 0층은 발견만 있고 철학적 앵커가 없다.
6. **신경현상학(바렐라)** — 열린 문제 #4에 "방법 있음, 미적용"으로 이미 등재 —
   네트워크에 점선 후보 노드로 걸어두면 열린 문제와 네트워크가 연동된다.

## 6. 우선순위 제안

1. **§1 (JSON 기반 자동 렌더 + 회차 슬라이더)** — 비용 대비 효과 최대. 이후 모든
   회차 확장이 공짜가 되고, #27 전파 문제도 구조적으로 해결.
2. **§2·§3 (층위 밴드 + 이원 회계 시각 구분)** — §1 위에 얹는 표현 계층. 특히
   이원 회계 구분은 정직성 요건.
3. **§4 (서브맵 분할)** — 노드 수가 감당 안 될 때의 후속 조치.
4. **§5 (새 개념 등재)** — 별도 검증 회차가 필요한 다른 성격의 작업. 네트워크
   구현과 독립적으로 진행 가능.
