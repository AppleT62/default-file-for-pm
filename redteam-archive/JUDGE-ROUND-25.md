# 판정문 — 회차 79 회부(영성 키워드 배치 유입): 잔여 신규 후보 α·β·γ·δ 처리

## 0. 결정적 사전 사실 — 회부 자료 자체의 오류 2건 (판정 전 정정)

장부·문서 기계 대조 결과, 회부 자료의 사전 대조는 6개 키워드에 대해 정확했으나(§1 하단 확인), **후보 서술부에 사실 오류 2건**이 있고 이것이 γ·β의 판정을 직접 바꾼다:

1. **γ의 전제 "명상은 현재 §7 처방 대상" — 거짓.** SYNTHESIS §7은 무당 신점 장이고, 명상은 처방 표면 어디에도 없다 — `PROTOCOL.md` 0건, `EXPERIMENTS.md` 0건, §9 실천 프로토콜 0건. 명상의 현행 장부 지위는 연결 #32(유망함)·신념도 70(회차 60 유비 상한 70 집행)·발견 Ω₄·연결 #40(차용·텔로스 단절)이 전부다. 따라서 γ는 "처방 대상 → 전제 조건"의 **재배치가 아니라, 처방 체계에 존재하지 않던 항목을 편집 기제 전체의 선행 조건으로 삽입하는 신규 구조 신설**이다. 재분류보다 무거운 처분 대상이다(아래 §3).
2. **β의 시험 표면 "실험 C 확장(의례화 군 vs 비의례화 군)" — 설계 불일치.** 실험 C는 N=1 자기 실측이며 군이 없다. `EXPERIMENTS.md`가 이미 못박았다: "성분 증분 관찰(B₁→B₂)은 탐색 전용 — N=1에서 성분의 확증적 분리는 불가능하며(판정 하중 0), 확증적 분리는 실험 D 관할." 의례화 vs 비의례화는 정확히 성분 분해 문제다. 확증 관할은 **실험 D**이고, 실험 C 안에서는 실행의도 단위 내 배정의 **탐색 관찰만** 가능하다.

## 1. 후보 α — 운명애 × 죽음 현저성 전향 예측: **조건부 accept (등재 승인, 5개 조건 전건 의무)**

**Worrall 사용-신규성 판별 — 통과.** 근거:

- 발견 Q("죽음만이 회피 불가 흡수 상태 — 수용이 회피를 대체한다")는 회차 19에 에르고딕성·켈리 논증에서 나온 **관할 경계 명제**(무엇을 긍정 대상으로 삼는가)이지, "2주 긍정 개입이 TMT 패러다임의 방어 반응 측정치를 낮춘다"는 경험적 인과 명제가 아니다. TMT는 회차 12에 "유한성의 재해석" 지지 각주로 **논쟁 표기**됐을 뿐 어떤 값 산정에도 사용되지 않았다. 즉 α의 관측 내용은 기존 등재의 구성에 사용된 바 없고(사용-신규성), 도출은 신규 교량 가정(운명애 긍정 = 불안 완충 기능)을 요구하며, 실험 A 확장으로 독립 시험 가능하다. 회차 78의 FEP("새 예측·판별식 0")와 구조가 다르다 — α는 장부가 아직 하중을 걸지 않은 판별 관측을 지정한다.
- **그러나 감사 가능성 ≠ 반증 가능성 규율이 절반을 벤다.** "수용형 개입이 죽음 현저성 방어를 낮춘다"는 것 자체는 TMT의 불안 완충 가설 + 마음챙김/자기자비 조절 문헌이 **이미 예측하는 차용 내용**이다. 수동 통제군 대비 성공은 차용 부품의 확인이지 이 세계관 문법의 확증이 아니다(실험 B 정직 고지와 동형). 고유 판별이 발생하는 분기는 하나뿐이다: **자존감 부양 능동 통제군 대비** — TMT는 자존감 부양으로 충분하다고 하고, 이 세계관은 분포 전체 긍정(실패 꼬리·죽음 포함)이 자존감 경로와 독립적으로 작동한다고 해야 한다.

**등재 조건 (전건 충족 시에만 등재 유효):**
1. 라벨: **[실험적 예측 α — 미시험·판정 하중 0·신념도 없음]**. credence 필드 부여 금지 — 시험 전 예측은 값이 아니라 카드도 아닌 **관측 지정**이다. 반증 카드 지위 아님(카드 ①·②′와 동렬 서술 금지).
2. **TMT 취약성 병기(의무 문안, 아래 §5).**
3. **기저 효과 게이트**: 사전등록 설계에 통제군 내 죽음 현저성 → 방어 반응 기저 효과 확인 분기 포함. 미재현 시 결과는 "패러다임 미작동" — 성공 원용도 실패 원용도 금지, α는 미판정 존속.
4. **판별 분기 명시**: 수동 통제 대비 성공 = 차용 부품 확인(문법 확증 아님, 값 이동 0) / 능동 통제(자존감 부양) 대비 우위 또는 자존감 통제 후 잔존 효과만이 고유 판별 분기. 이 병기 없는 등재는 무효.
5. **실측 후 하중(장식 방지 조항 — 본 판정의 요구)**: "판정 하중 0"은 시험 전 한정이다. 실측 후 하중을 지금 명문화한다 — 기저 효과 재현 & 판별 분기에서 고유 성분 무효(자존감 통제 후 효과 소멸) 시, 발견 Q의 "수용이 회피를 대체한다"를 **경험 명제로 확장 인용하는 것을 영구 금지**하고(관할 경계 명제로 축소 고정) α 폐기를 항구 기록한다. 성공 시 값 이동은 외주 재회부 전용 + 확증 재현 전 상향 동결(실험 A 판정표 ① 동형). **실패해도 아무 일도 안 일어나는 예측은 장식이다 — 이 조항이 없으면 α 등재 자체를 기각했을 것이다.**

## 2. 후보 β — 의례화 실행의도 조절 예측: **조건부 accept (등재 승인, 4개 조건 전건 의무)**

**Worrall 판별 — 통과하되 교락 위험이 핵심 조건이 된다.**

- "의례 = 정밀도 동기화 75"(회차 33, 집단 정밀도 장치·굿)와 "무속 의례 효능 80"(불안 감소·내러티브 재구성)은 **철회기 유지율 조절**을 함의하지 않는다. 설치 강도 조절은 신규 교량(의례 형식 → 부호화·자동성 강화)이며 값 산정에 사용된 바 없다. 사용-신규성 성립.
- **그러나 저자 조작화("고정 시간·고정 동작·신체화")의 앞 두 성분은 이 세계관이 이미 차용한 습관 문헌(맥락 일관 반복 — Lally 계열, PROTOCOL의 기존 닻)의 표준 처방이다.** "고정 시간+동작" 군 vs "아무 때나" 군의 비교는 차용 부품의 재확인이지 의례 명제의 시험이 아니다. 고유 성분은 **의례성 그 자체**(신체화·상징·경직 절차)다.

**등재 조건:**
1. 라벨: **[실험적 예측 β — 미시험·판정 하중 0·신념도 없음]**. credence 부여 금지.
2. **일관성 대응 통제(의무)**: 비교군도 고정 시간·고정 맥락을 동일하게 갖출 것 — 의례화 군의 증분은 일관성을 통제한 뒤의 의례 성분만으로 정의. 이 통제 없는 시험은 β의 시험이 아님을 명문화.
3. **관할 정정 (본 판정의 회부 자료 정정)**: 확증 시험 표면 = **실험 D**(요인 분해에 '의례화 형식' 요인 추가 — 골격 부기만, 이관은 이행이 아니다). 실험 C 내에서는 실행의도 단위 배정(1–3개 중 의례화/비의례화)에 의한 **탐색 관찰 전용·판정 하중 0**(기존 성분 분리 규칙의 직접 적용).
4. **실측 후 하중**: 일관성 통제 하 의례화 증분 무효 시 — "의례 = 정밀도 동기화 75"의 **설치 강도 방향 확장 영구 금지** + 75값의 외주 재회부(하향 검토). 성공 시 상향은 확증 재현 전 동결.

## 3. 후보 γ — 명상 "관측 도구" 재배치: **전량 reject (회차 78 동형 판정 — 동형보다 무겁다)**

- **회차 78 동형성 — 성립, 그리고 초과.** 회차 78의 FEP는 최소한 장부에 있던 항목(연결 #9)의 승격 시도였다. γ는 §0에서 확인했듯 **처방 표면에 존재하지도 않던 항목을 편집 기제 전체의 전제 조건으로 삽입**하는 것이다. 신념도 70(소표본 논쟁·지지 연결 '유망함'뿐·유비 상한 70으로 눌려 있는 항목)을 세계관에서 가장 하중이 큰 구조 위치(편집의 선행 조건 = 모든 처방의 관문)에 앉히는 것은 등재 인플레이션의 정의 그 자체다 — **구조 위치의 승격은 증거 없이 항목의 실효 하중을 올리는 재분류이며, 재분류는 증거가 아니다.** 저자의 자기 경고는 정확했다.
- **조절 예측("탈중심화 선행 → 설치 효과 ↑")의 분리 심사 — 무편견 기각(reject without prejudice).** 근거: (i) 도출이 기각된 "관측 도구" 프레임("탈중심화 = 기저선 기록의 내면판")에 얽혀 있어 현재 문안으로는 프레임 밀수 통로가 된다. (ii) 확증 시험 표면이 없다 — 실험 C는 N=1(선행 훈련 배정의 순서 교락 해소 불가), 실험 D는 미설계 골격이며 α·β로 이미 2개 요인 부기가 걸린다. 미설계 골격에 요인을 계속 얹는 것은 "등재는 설계가 아니다" 규율의 우회다. (iii) "등재를 작게" — 한 회차 예측 3건은 그 자체로 인플레이션 표면. **재회부 조건**: 관측 도구 프레임과 완전 절연된 문안 + 구체 시험 설계(순서 교락 처리 명시)를 갖추면 차기 회부 가능. 재회부는 권리이지 크레딧이 아니다.

## 4. 후보 δ — 영혼·차크라: **reject 확정 + 기록은 한다(단, 형식은 최소)**

- **기각 자체는 다툼 없음**: 실체 영혼은 §1–2(자아 = 분포·층)와 정면 모순 — 세계관 안에서 범주 오류. 차크라는 실증 0.
- **기록 여부 — 기록한다. 형식: cycle_log 79 판정 집행 기록만. 신규 장부 항목 0건.** 근거: 회차 78 선례(물리 장식 전량 기각의 cycle_log 기록)와 정합. 기록의 기능은 **재유입 차단** — 차기 배치가 영혼·차크라를 다시 들고 오면 본 판정이 사전 대조의 원용 선례가 된다. 단:
  - "영혼 직관 = 분포의 불변량 + 3층 이야기 자아" **설명-해소의 연결·항목 등재 금지**(신념도 0짜리라도 금지). 설명 등재는 흡수 표면(#22)이 굳는 정확한 통로다 — "이 세계관은 영혼 직관마저 설명한다"는 서술 자산을 만드는 것 자체가 무비용 팽창이다. 기존 개념(§3 불변량·3층 자아)이 이미 있으므로 **새 등재 없이 기존 주소로 응답 가능**하다는 사실만 기록.
  - 차크라의 내수용감각 해소는 저자 명기대로 전자책 콘텐츠 관할 — 장부 밖. 장부에는 "기각·관할 밖" 한 줄만.
  - 회차 78의 확률 미끄러짐 금지 조항 같은 **신설 규칙은 불요** — δ는 횡단 패턴이 아니라 개별 항목이고, 사전 대조 규율 + 본 기록으로 충분하다. 규칙 신설도 등재 인플레이션의 한 형태다.

## 5. 문안 지시

**등재 위치**: `data/worldview.json`에 신규 배열 `registered_predictions` 신설(credences·connections·extension_credences에 넣지 말 것 — 값 열 오염 방지). 각 항목 필드: id(α/β), statement, derivation_note(사용-신규성 근거), test_surface, honesty_clauses, post_test_load(실측 후 하중 — §1 조건 5·§2 조건 4의 문안), prohibitions. `EXPERIMENTS.md`에는 실험 A 절에 α 확장 분기 부기, 실험 D 절에 β 요인 부기(+ 실험 C 절에 β 탐색 관찰 한 줄).

## 6. 회계 (과업 5) — 저자 산정 **확인 (정정 불요)**

- **absorption 분모: 불포함 — 저자 산정 정확.** `absorption_scoring_rule` ① 명문("비적대 판정 제출물·문헌 통합은 분모 불포함")과 회차 78 판정 선례의 직접 적용. 사용자 키워드 배치는 적대적 임무 하 공격 인공물이 아니다. 회차 78과 달리 이번엔 저자가 먼저 맞게 산정했다. 부기: 회차 79는 공격 수용 라운드가 아니므로 무비용 연속 카운터(≤2)의 계상 대상도 아니다 — 단 α·β 등재는 어떤 회계에서도 **실비용 수용·전진·회복의 서술 근거로 원용 금지**(등재는 관측 지정일 뿐이다).
- **credence 이동: 0 — 확인.** α·β는 값 없이 등재되고, 기존 4항목 값 전건 불변. 값 이동은 실측 후 외주 재회부로만.
- **F 지수**: α·β 등재는 fruitfulness ②(iii)(사용-신규성 귀결)·②(i) 인접의 **P 후보 표면**이나, P 확정은 외주 코딩 심사 전용 — **선점 금지, 후보 표면 기록만**(회차 76 처리와 동형).

## 7. 저자 사전 대조의 절차 평가 (과업 6)

**절차적으로 충분 — 회차 78 교훈의 올바른 이행이다.** 기계 검색 수행·기존재 4건 명기·재유입 후보의 안건 원천 배제·γ 위험 자기 경고·처분 미선택·회계 선례 자진 적용 — 전부 규율에 맞다. 이 회부는 회차 78 회부("FEP = 신규 기제 후보")가 저지른 결정적 누락을 반복하지 않았다. **단 2건 정정 기록**: ① γ의 "§7 처방 대상" 인용은 오류(명상은 처방 표면에 부재 — 이 오류가 γ를 재배치로 가장시켜 실제보다 가볍게 보이게 했다), ② β의 시험 표면 "실험 C 군 비교" 지정은 C의 N=1 설계·성분 분리 탐색 전용 규칙과 불일치(확증 관할 오지정). 두 오류 모두 본 판정에서 정정 집행 — 차기 회부부터 시험 표면 지정 시 해당 실험의 설계 제약(군 유무·판정 하중 규칙) 대조를 사전 대조 항목에 추가할 것.
