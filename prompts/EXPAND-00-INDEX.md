# 세계 편 확장 배치 — 프롬프트 묶음 안내

이 저장소의 세계관("운명 = 조건부 확률 분포, 자유 = 유계적 편집")은 **"나"의 작동원리는 79회차에 걸쳐 깊게 팠으나 "우리"(분포들의 상호작용·시스템)는 거의 미탐사**다. 이 묶음은 그 공백을 채우는 6개 배치의 다른 세션용 프롬프트다.

## 사용법

1. 한 번에 **한 배치만** 진행한다(등재 인플레이션 방지 — 이 프로젝트의 규율).
2. 해당 MD 파일을 새 세션(Claude 또는 Gemini Deep Research)에 **그대로 붙여넣는다**(각 파일은 자체 완결).
3. 산출물(MD)을 원 저장소로 가져온다.
4. 원 저장소 세션이 **표준 프로토콜**로 처리: 장부 사전 대조(재유입 차단) → 회부 자료 작성(저자 판별 없음) → **독립 판정 세션에 외주** → 판정 그대로 집행 → 검증 → PR·머지.

## 공통 규율 (6개 전부에 박혀 있음)

- **개념 장식은 기각된다.** 가치 있는 것은 "이 세계관이 틀렸다면 실패할 새로운 예측"뿐(회차 78·79에서 확립).
- **재유입 금지**: 장부에 이미 있는 것(스케일 불변성·마태 효과·아비투스·M2 등)을 다시 등재하지 않는다.
- **은유/기제 구분 표기**, 출처·재현 논쟁 병기, 반례 수집 의무.
- **없으면 없다고 하라** — 억지 예측 금지(특히 ⑤).

## 추천 진행 순서

**① 협력**(세계 편 초석·문헌 최강) 또는 **⑥ 방법론 장**(탐사 없이 지금 바로 집필 가능·소유자의 "가치관 만드는 법" 목적 직결)부터. ⑥은 판정 회부 불요라 가장 빠르게 완결된다.
