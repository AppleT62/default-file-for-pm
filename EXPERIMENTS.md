# 고유 명제 실험 설계 — 반증 카드의 실험화

> 회차 44에서 신설한 고유 명제 반증 카드를 실제 실험 설계로 구체화한다(회차 48).
> 카드 이력(산술 그대로): 2장 → 1장(② 철회, 회차 69 — 조작 미정의·표적 상실) →
> 2장(②′ 재작성, 회차 72 — 외주 18회차 문언 확정, **미시험**). 철회 기록은 불소멸이다.
> 카드들이 실험으로 시험되기 전까지, red-team 2차 판정 "(c) 반증불가 유사이론 쪽으로
> 1노치 악화"는 유효한 혐의로 남는다 — 이 문서는 그 혐의를 벗을 수 있는 유일한 경로의 설계도다.
>
> 지위: **설계 초안.** 사전등록(OSF 등) 없이 수행된 결과는 카드 촉발/해제의 근거로
> 인정하지 않는다(자기 채점 편향 차단 — RED-TEAM.md와 같은 원리).

## 실험 A — 운명애 루프 (기제 가설 M1 · 발견 C — 회차 56 갱신)

**시험하는 고유 명제**: "자기 분포 전체(실패 꼬리 포함)에 대한 긍정은 자기방해 행동을
줄인다" — 4고리 사슬 중 이 세계관이 스스로 조립한 핵심 고리(긍정 → 자기방해 차단).
성과 사슬 전체가 아니라 최약 고리 하나를 직접 시험한다(회차 44 결합 규칙, 회차 47
원산지 정정 — 루프는 성과 약속이 아니라 자기방해 차단 기제로만 읽는다).

- **설계**: 사전등록 무작위 배정, 2군 병렬 + 조절변수 1개.
- **조작**: 개입군 — 2주간 격일 "분포 긍정" 글쓰기(실패 사례를 '나의 분포의 정상 꼬리'로
  재기술 + 회복 가능성 확인, 회기당 10분). 통제군 — 동일 분량의 중립 기술 글쓰기(일과 기록).
- **1차 결과변수**: 자기방해(self-handicapping) — ① 척도(SHS) 사전/사후, ② 행동 지표:
  중요 과제 전 연습 시간 선택 패러다임(자기방해의 고전 조작화 — 연습 회피 = 자기방해).
- **조절변수(사전등록)**: 방어적 비관주의(Norem & Cantor DPQ). 회차 43 [경미13]의 경계
  조건을 검증 항목으로 승격 — 방어적 비관주의 상위군에서 개입이 역효과인지.
- **표본**: 군당 n≈180 — d=0.30, 검정력 80%, α=.05 양측 기준. 회차 56의 도구화 탐색 조건군 추가로 **3군 설계 — 총 ≈540, 중도 탈락 20% 가정 시 모집 ≈675**(회차 57 재산정: 구판 "총 360·모집 450"은 2군 기준 화석 — RT-1 적발). 실행 주체: 이 규모는 연구자·기관용이다 — 이 저장소의 개인 독자는 실험 C가 당신 몫이다.
- **분석**: 사전-사후 변화량에 대한 군간 비교(ANCOVA, 사전값 공변량), 조절효과는
  군×DPQ 상호작용.
- **판정 규칙 (회차 50 재작성 — red-team α[B-2·B-3]: 구판은 이론이 정확히 옳아도 20%
  확률로 공리를 폐기하는 규칙이었고, "효과 없음"을 비유의로 판정하는 오류가 있었다)**:
  - **1차 변수는 행동 지표 하나로 지정**(연습 시간 선택 패러다임). SHS 척도는 2차 변수
    (α 보정, 판정 하중 없음).
  - **조작 점검 신설**: 개입 후 상태 운명애/수용 척도로 유도 성공을 확인한다. 조작 점검
    실패 시 결과는 "조작 실패"이지 이론 반증이 아니다 — 카드는 켜지지 않고 재설계로 간다.
  - **촉발(폐기)** = 등가성 판정: 1차 변수 효과의 90% CI 상한이 d = 0.15 미만
    (TOST 등가 한계 ±0.15). 이 판정에 필요한 표본은 군당 ≈550 — 기본 표본(180/군)으로는
    촉발 판정이 **불가능**함을 명시한다. 기본 표본의 비유의는 "판단 유보"다.
  - **판정표 (전 구간 — 회차 76 개정, 외주 22회차: ④ 재라벨·①의 +5 동결·⑤ 신설)**:
    ① 유의 & d̂ ≥ 0.30 → 값 유지. **+5(값 상향)는 확증적 재현 1회 AND 그 재현이 등가 판정
    가능 표본(≈550/군)에서 수행됐을 것을 모두 충족하기 전까지 동결**(단일 연구 상한 + 비대칭
    제거 — 기본 표본 단계에서 카드는 켜질 수만 있고 값을 올릴 수는 없다).
    ② 유의 & d̂ ∈ [0.15, 0.30) → 질적 생존, 값 유지. ③ 비유의 & CI 상한 ≥ 0.15 →
    판단 유보(카드 미촉발, 증거 불충분 기록). ④ CI 상한 < 0.15 (등가성 입증) → **카드 촉발**:
    기제 가설 M1·발견 C 폐기, M1 30 이하. **대표본 전용 조항 — 군당 ≈550 필요, 기본 표본
    (180/군) 촉발 불가. 본 분기의 존재는 장치 실적 규율상 반증 가능성의 증거로 원용 불가.**
    ⑤ **(신설, 회차 76)** 유의 & **반대 방향**(운명애 긍정군의 자기방해가 통제군보다 유의하게
    증가, α=.05 양측 반대 꼬리) → **카드 촉발**: M1·발견 C 폐기, M1 30 이하. **기본 표본
    (n≈180/군)으로 관찰 가능·자동 발동·본인/저자 세션 재량 불가·조작 점검 항변 부적용(면역
    장치 봉쇄) — 이론이 옳을 때 오촉발 ≤ α/2 ≈ 2.5%.**
  - 방어적 비관주의 상위군의 유의한 역효과(군×DPQ 상호작용) → 폐기가 아니라 **경계
    조건을 M1 본문에 승격**. 단 상호작용 검정은 주효과 대비 ~4배 표본이 필요하므로
    이 결과는 탐색적 표기만 가능함을 사전 명시.
  - 분석은 ITT 원칙, 평정자 맹검(행동 지표 코딩), 피험자 비맹검은 한계로 사전 등재.
  - **도구화 탐색 조건군(회차 56, 외주 판정 승인 — 탐색 한정)**: 제3군(긍정 글쓰기 +
    성과 기대 유도)을 추가해 M1의 작동 조건("의도적 도구화 시 효과 소멸")을 시험한다.
    사전등록에 탐색 표기, 판정 하중 없음 — 이 조건에서 효과가 유지되어도 역설 주장은
    '폐기 후보'로 강등될 뿐이며, 폐기는 확증적 재현 1회를 요구한다(단일 연구 상한 규율).

**반증 카드 ①(실험 A / 기제 가설 M1·발견 C — 회차 76 촉발 문턱 정비, 외주 22회차 문언 확정.
#21 술어 관할)**: **표적** = "회복 가능한 자기 분포 전체(실패 꼬리 포함)에 대한 긍정은
자기방해를 줄인다"(M1·발견 C — 이 세계관이 스스로 조립한 핵심 고리). **촉발 조건(기계적) —
두 경로**: 【기본 표본 경로】 사전등록 실험 A에서 1차 변수(자기방해 행동 지표 — 연습 시간 선택
패러다임)가 예측과 반대 방향의 유의 효과(운명애 긍정군의 자기방해가 통제군보다 유의하게 증가,
α=.05 양측). 기본 표본(n≈180/군)으로 관찰 가능. 【대표본 경로】 TOST 등가 입증(90% CI 상한
< d=0.15) — 군당 ≈550 필요, 기본 표본(180/군) 촉발 불가함을 명시(대표본 전용 — 그 존재는
장치 실적 규율상 반증 가능성의 증거로 원용 불가). 조작 점검 조항은 기본 표본 경로(반대 방향
유의)에 부적용 — 조작 실패 항변으로 촉발을 무효화할 수 없다(면역 장치 봉쇄). **발동(자동·
본인/저자 세션 재량 불가)**: ㉠ M1·발견 C 폐기 — M1 30 이하(이력 보존형 철거), ㉡ 운명애 루프
60 장부 항목(회차 56 M1 병합)의 잔여 가격 동일 회차 외주 재회부 — 기본값 하향/철거, ㉢ §5
태도 층·부록 2 M1 문단·도구화 역설 '작동 조건'의 지위 재서술·재회부, ㉣ M1 의존 처방 표면
(PROTOCOL·§9) 갱신, ㉤ M-상한·연동 산식의 열 정합 재확인. **촉발 확인 절차**: 결과 공표 시
독립 판정 세션 회부 — 판정 대상은 촉발 성립의 기계적 확인(사전등록 정합·유의성·방향)뿐,
촉발 회피 재량 없음. (회차 44 신설 카드 ①의 문턱 현실성 정비 — 카드 ②′(회차 72)와 함께
#21 술어 전반부 '전 종 재작성' 완주. 단 미시험 재작성 — 반증 태세 회복 아님, '회복 시도(미시험)'
까지만 서술 가능.)

**예측 α 확장 분기 부기 (회차 79, 외주 판정 25회차)**: 본 실험의 확장 설계로 죽음 현저성
조작 부가 분기가 등재됨 — [실험적 예측 α — 미시험·판정 하중 0·신념도 없음] "운명애 개입은
죽음 현저성 유도 후의 방어 반응을 통제 대비 낮춘다"(전문·조건 5건: `data/worldview.json`
registered_predictions). 핵심 조건: TMT 재현 논쟁 병기·기저 효과 게이트(미재현 시 '패러다임
미작동' — 성패 원용 쌍방 금지)·판별 분기(수동 통제 성공 = 차용 확인·값 이동 0, 고유 판별은
자존감 부양 능동 통제 대비만)·실측 후 하중(고유 성분 무효 시 발견 Q 경험 확장 인용 영구
금지 + α 폐기 항구 기록). 등재의 반증 가능성 증거 원용 불가.

## 실험 B — 온도 축 (공리 V · 발견 S)

**시험하는 고유 명제(회차 72 재지정, 외주 18회차)**: "개인 규모에서, 산출 분포의
의미적 다양성은 온도 조작(판단 유예+무관 자극 노출+시간 압박 제거의 복합 조작)에
예측 방향(고온 → 다양성 증가)으로 반응한다" — 발견 S '조작 가능한 축' 주장의 규모 특정
조작화. 구 표적(공리 V 폐기·'한 규모에서라도 반대 효과')은 3중 실효(공리 V 축약 회차 64·
카드 ② 철회 회차 69·스케일 불변성 철거 회차 70)로 삭제 — 이력 보존. 규모 간 이전은
본 실험의 시험 대상이 아니다.

**반증 카드 ②′(회차 72 재작성, 외주 18회차 문언 확정 — #21 술어 관할. 카드 ② 철회(회차 69)의
대체 등재, 철회 기록 불소멸)**: **표적** = 위 재지정 표적 하위 명제. 공리 V·스케일 불변성은
본 카드의 표적이 아니다(각각 축약·철거 상태). **촉발 조건(기계적)** = 사전등록된 실험 B에서
1차 변수(의미적 다양성 — 사전 명시된 계산 절차)가 예측과 반대 방향의 유의 효과(α=.05 양측).
촉발은 기본 표본(n≈66)으로 관찰 가능하다. 조작 점검 조항은 촉발 분기에 부적용 — 조작 실패
항변으로 촉발을 무효화할 수 없다(면역 장치 금지). **발동(자동·본인/저자 세션 재량 불가)**:
㉠ 표적 하위 명제 폐기, ㉡ 발견 S(자유의 두 번째 축)의 온도 편집 성분 폐기 — 확장 장부 항목은
내용 편집(실행 의도) 성분만으로 재서술·값 재산정 외주 회부, ㉢ 편집 기제 65 scope_note에
'온도 성분 비보증' 추가, ㉣ 온도 축 의존 확장 항목 4건(창의성=온도의 2차 운용·발산/수렴=온도
대응·놀이=안전한 온도 탐색·편집 4상태)의 잔여 가격 동일 회차 외주 재회부 — 기본값 철거(이력
보존형), ㉤ PROTOCOL·§9의 온도 편집 인용 처방 동일 회차 갱신(처방 연동 규칙 ③).
**촉발 확인 절차**: 결과 공표 시 독립 판정 세션 회부 — 판정 대상은 촉발 성립의 기계적
확인(사전등록 정합·유의성·방향)뿐, 촉발 회피 재량 없음.

- **설계**: 사전등록, 피험자 내 교차(within-subject, 순서 상쇄).
- **조작**: 동일 인물이 두 조건에서 아이디어 과제 수행 — 고온 조건(판단 유예 지시 +
  무관 자극 노출 + 시간 압박 제거), 저온 조건(명시적 평가 기준 + 선별 지시).
- **1차 결과변수**: 산출 아이디어의 ① 의미적 다양성(임베딩 분산 등 사전 명시된 계산
  절차), ② 독창성/실행가능성 평정(맹검 평정자 2인).
- **예측(사전 명시)**: 고온 → 다양성·독창성 ↑, 실행가능성 ↓. 저온 → 역방향.
- **표본**: n≈66 (피험자 내 dz=0.35, 검정력 80% — 회차 50 정정: 구판 n≈60은 검정력 ~76–77%로 계산 불일치(red-team α[B-4]·γ 동시 지적). t-보정 포함 66–67명).
- **판정 규칙 (회차 72 전면 개정, 외주 18회차 — 구판 회차 50 규칙의 실효 참조 청소)**:
  ㉠ 1차 변수(의미적 다양성) 반대 방향 유의 → **카드 ②′ 촉발**(카드 문언의 연쇄 발동).

## 실험 C — 66일 자기 실측 (N=1, PROTOCOL.md)

기존 `PROTOCOL.md`가 도구다. 실험으로 격상하는 설계(회차 72 개정, 외주 18회차):

1. **단계 도입 설계(A–B₁–B₂–A′)**: 기저선 2주(기록만) → B₁ 3주(+실행 의도) →
   B₂ 3주(+수면 게이트 적극 운용·중요도 낮추기) → 철회 2주(기록만).
   철회기에 행동이 기저선으로 즉시 복귀하면 "설치"가 아니라 "노력 유지"였던 것 —
   Hudson 외(2019)의 완수 조건과 대조 가능. **성분 증분 관찰(B₁→B₂)은 탐색 전용** —
   N=1에서 성분의 확증적 분리는 불가능하며(판정 하중 0), 확증적 분리는 실험 D 관할.
   판정은 전체 개입 vs 기저선·철회 복귀 기준 불변. **B₂ 개시 전 임상 게이트 재확인
   의무**(방어적 비관주의 역효과 경계의 직접 적용 대상). (회차 79 부기: 실행의도 단위
   배정에 의한 의례화/비의례화 관찰(예측 β)도 동일 규칙 — **탐색 전용·판정 하중 0**,
   확증은 실험 D 관할.)
2. **판정의 외주**: 시작 전에 성공/실패 기준을 수치로 적어 두고, 66일 후 판정을
   본인이 아닌 제3자(또는 다른 세션의 AI)에게 맡긴다 — 감사자=피감사자 문제의 개인판 차단.
   판정 외주 계약에 **안전 중단 규칙 준수 여부의 확인 항목**을 포함한다(외주 18회차 소견 3 —
   규칙의 존재는 원용 불가하고, 확인된 준수만이 기록이다).
3. **안전 중단 규칙(회차 72, 보고서 #5 G6 — 자동 발동·본인 재량 불가)**:
   ㉠ 주간 체크에서 기분·수면·기능 중 **2개 영역 이상의 악화가 동시 관찰되면 즉시**
   개입 중단 + PROTOCOL 임상 게이트의 전문 평가 절차로 이행. ㉡ 어느 1개 영역이라도
   악화가 **2주 연속**이면 개입 중단(시작 전 게이트의 '2주 지속' 문턱의 운용 중 적용).
   ㉢ 본 규칙은 하한 — 기존 조항(재측정부: 악화 체크 시 중단)이 더 보수적이면 그쪽이 우선.
   해당 조건 관찰 = 중단이며, 계속 사유의 기록은 허용하되 효력 없음. ㉣ 재개는 전문 평가 후
   이상 소견 없음 확인 또는 철회기 진입으로만 — 재개 사실·사유 기록 의무. ㉤ 기록 부담
   문항(#24)이 2주 연속 상위 앵커면 기록 부하를 일 1건으로 감축(중단 아님 — 부담 완화).
   감축은 프로토콜 변경 — 발생 시점·사유 기록·판정 시 명기. ㉥ 중단·감축은 실패가 아니라
   안전 게이트의 실작동으로 기록하며 #24의 관측 자료다. 단 **중단된 실험 C는 판정 미완** —
   중단 사실을 이론 지지 증거로도 신뢰성 실적의 추가 논증으로도 원용 금지.

이것은 이론 전체가 아니라 **당신의 분포에서 편집 기제가 작동하는가**만을 시험한다 —
